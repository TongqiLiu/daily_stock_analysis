# 查询实时贪恐

> 创建人: V

> 更新人: V

> 创建时间: 2024-12-02 23:14:37

> 更新时间: 2025-09-05 22:47:16

**接口基地址：https://szdt.tech**

**接口状态**

> 已完成

**接口URL**

> /api/partner/invest/stock/scan

**请求方式**

> POST

**Content-Type**

> form-data

**请求Header参数**

| 参数名 | 示例值 | 参数类型 | 是否必填 | 参数描述 |
| --- | --- | ---- | ---- | ---- |
| X-Auth |  | string | 是 | 会员激活码 |

**请求Body参数**

| 参数名 | 示例值 | 参数类型 | 是否必填 | 参数描述 |
| --- | --- | ---- | ---- | ---- |
| code | US.YINN | string | 是 | 完整股票代码，例如 SH.510300、SZ.000001、HK.00700、US.TQQQ |
| lever | 3 | string | 是 | 杠杆倍数，1~4 |
| emo_area | a | string | 是 | 资产类型，a表示中概，us表示美概，coin表示币股，other表示其他 |

**认证方式**

> 继承父级

**响应示例**

* 成功(200)

```javascript
{"status":1,"msg":"成功","data":{"query":2,"query_api":4,"name":"3倍做多纳指ETF-ProShares","price":82.07,"score":71,"time":"2024-12-02 23:25:07"}}
```

| 参数名 | 示例值 | 参数类型 | 参数描述 |
| --- | --- | ---- | ---- |
| status | 1 | number | - |
| msg | 成功 | string | - |
| data | - | object | - |
| data.query | 2 | number | 已查询个数，会员每 30天 免费 30 个，重复查询同一只股票只扣 1 个，滚动释放 |
| data.query_api | 4 | number | 已查询次数，会员每自然月免费 5000 次。贪恐 -30 ~ 30 的股票推荐设置 redis 缓存，可以大幅减少请求次数 |
| data.name | 3倍做多纳指ETF-ProShares | string | 股票名称 |
| data.price | 82.07 | number | 当前价格 |
| data.score | 71 | number | 贪恐指数 |
| data.time | 2024-12-02 23:25:07 | string | 当前北京时间 |

* 其他错误(200)

```javascript
{"status":2,"msg":"杠杆倍数错误","data":[]}
```
